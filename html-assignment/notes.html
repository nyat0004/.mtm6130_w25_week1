HTML Tags and Descriptions

h1 - h6
These tags define headings, with h1 being the largest and h6 the smallest.
p
Used to create paragraphs in a webpage.
img
Inserts an image into a webpage using the src attribute.
a
Defines a hyperlink, allowing users to navigate to another page or website.
div
A container element used to group and style content on a webpage.
